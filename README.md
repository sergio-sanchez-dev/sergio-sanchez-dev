## Proyecto principal

> **App real en producción** — no un proyecto de academia.
> Desarrollada para CasaBlanca Inmobiliaria durante prácticas (marzo–mayo 2026).

### 📱 [CasaBlanca Inmobiliaria App](https://github.com/TU-USERNAME/casablanca-inmobiliaria-app) `producción`

App Android completa de gestión inmobiliaria en campo.
Digitaliza el 100% del flujo de visitas, captaciones, ventas y contratos.

**Lo que hace:**
- Autenticación con Firebase Auth y sistema de roles por perfil de usuario
- Consumo de APIs REST para sincronización con backend
- Generación automática de PDFs con firma electrónica integrada
- Sincronización de tareas y recordatorios con Google Calendar
- Gestión de imágenes y documentos desde el móvil

`Java` `Kotlin` `Firebase Auth` `Firestore` `REST API` `PDF + firma digital` `Google Calendar API`
